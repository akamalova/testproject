# testproject
Test project for JavaRush

1. Запустить MySQL на localhost:3306 с логином и паролем root;
2. Создать схему test;
3. Запустить скрипт BookSkript.sql;
4. Открыть и запустить в IDE KamalovaTestProject;
5. Создать новый проект в Angular CLI (Install Angular CLI  https://angular.io/guide/quickstart) 
6. Заменить в созданном проекте Angular CLI папку src на загруженную из репозитория;
7. В командной строке набрать (мой проект называется my-app):
cd my-app
ng serve --open
8. Проект загрузится в браузере по ссылке http://localhost:4200;
